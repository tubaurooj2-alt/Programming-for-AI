# 🌿 Maze Quest

A jungle maze game with BFS/DFS AI, coins, traps, and sound effects.

## 🎮 Play Online

👉 **[Play Maze Quest](https://YOUR-USERNAME.github.io/YOUR-REPO-NAME/)**

## 🚀 How to Deploy to GitHub Pages

1. Create a new GitHub repository (e.g. `maze-quest`)
2. Upload these 3 files:
   - `index.html`
   - `maze_quest.html`
   - `README.md`
3. Go to **Settings → Pages**
4. Under **Source**, select `Deploy from a branch`
5. Set branch to `main` and folder to `/ (root)`
6. Click **Save**
7. Wait ~60 seconds — your game is live!

## 🕹️ Controls

| Key | Action |
|-----|--------|
| Arrow keys / WASD | Move |
| Space | Start game |
| R | New maze |

## 🔊 Sound Effects

- 🧱 Wall placed — tactile click
- 🤖 AI moving — rising blip
- 🎉 Win — triumphant arpeggio
- 🪙 Coin — bright ping
- 💀 Trap — harsh buzz
- ⏰ Time up — alarm tones
